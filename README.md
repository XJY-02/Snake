# 贪吃蛇游戏 - C++控制台实现

一个使用C++编写的控制台贪吃蛇游戏，包含完整的游戏循环、设置系统和排行榜功能。

## 项目简介
这是一个独立设计实现的贪吃蛇游戏，重点练习C++面向对象编程、多文件项目组织和游戏逻辑开发。游戏采用控制台界面，包含完整的游戏流程和可自定义的设置选项。

## 功能特点
-  完整的贪吃蛇游戏核心逻辑
-  可自定义游戏设置（地图大小、速度、穿墙等）
-  游戏数据统计与排行榜系统
-  随机种子控制，支持可复现对局
-  控制台图形化界面，蛇头/身/尾视觉区分

## 编译运行

### Windows (MinGW)
```bash
g++ -std=c++17 -I./include src/*.cpp -o build/snake_game.exe
.\build\snake_game.exe
```

### 其他平台
```bash
g++ -std=c++17 -I./include src/*.cpp -o build/snake_game
./build/snake_game
```

## 游戏操作
- **W/A/S/D** - 控制蛇移动方向
- **ESC** - 返回主菜单/退出游戏
- **Enter** - 确认选择

## 设计思路

### 面向对象设计
- **单一职责原则**：每个类专注于特定功能
  - `Game`：游戏流程控制
  - `GameBoard`：地图管理与渲染
  - `Snack`：蛇的移动与生长逻辑
  - `Food`：食物生成与碰撞检测
  - `GameStats`：数据统计与排行榜
  - `GameSetting`：游戏配置管理

- **组合模式**：`Game`作为主控制器，组合管理所有游戏组件
- **低耦合设计**：通过`friend`关键字控制类间访问，保持封装性

### 关键算法实现
1. **蛇移动**：使用`std::deque`实现高效的头插尾删操作
2. **食物生成**：随机生成并避免与蛇身、其他食物重叠
3. **碰撞检测**：实时检测墙壁、身体碰撞
4. **排行榜系统**：按得分和时间多条件排序

- 排行榜记录上限为3条
